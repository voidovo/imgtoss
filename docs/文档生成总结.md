## 🚀 部署和使用

### 快速启动
```bash
# 进入文档目录
cd docs

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 或使用自动化脚本
./deploy.sh preview
```

### 部署选项
- **GitHub Pages**: 使用提供的 GitHub Actions 工作流
- **Netlify**: 连接仓库自动部署
- **Vercel**: 使用 vercel.json 配置
- **自定义服务器**: 上传 dist 目录内容
